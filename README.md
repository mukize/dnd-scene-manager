# DnD Scene Manager

A scene manager for Dungeons and Dragon (or any ttrpg).
Building a scene manager for a friend to manage and display maps and interactions for their dnd campaigns.
Still WIP.

[x] Scene CRUDS
[x] Adding backgrounds to scenes
[x] Naming scenes
